## Architecture

```
┌─────────────────────────────────────────┐
│           Userspace (ioctl)             │
├─────────────────────────────────────────┤
│  chardev · debugfs · Control Plane      │
├─────────────────────────────────────────┤
│       Core Runtime (registry,           │
│       event bus, power manager)         │
├─────────────────────────────────────────┤
│  Backend Adapters                       │
│  (QEMU │ MMIO │ Platform │ DeviceTree)  │
├─────────────────────────────────────────┤
│       Device Drivers                    │
│  (UART │ I2C │ SPI │ GPU │ NPU)         │
├─────────────────────────────────────────┤
│  KDAL Language Toolchain                │
│  (.kdh/.kdc → kdalc → kernel module)    │
└─────────────────────────────────────────┘
```

## Quick Install

```sh
curl -fsSL https://raw.githubusercontent.com/NguyenTrongPhuc552003/kdal/main/scripts/installer/install.sh | sh
```

Or with a package manager:

```sh
# Homebrew (macOS / Linux)
brew install kdal-project/tap/kdal

# Debian / Ubuntu
sudo apt install kdal

# Fedora / RHEL
sudo dnf install kdal
```

## Target Platforms

| Platform              | Architecture | Status           |
| --------------------- | ------------ | ---------------- |
| Linux (QEMU aarch64)  | ARM64        | Primary target   |
| Linux (native x86_64) | x86_64       | Development host |
| Linux (RISC-V)        | riscv64      | Planned          |
| macOS (Apple Silicon) | ARM64        | Toolchain only   |
